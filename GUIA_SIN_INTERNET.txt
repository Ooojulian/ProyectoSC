ANTES DE EMPEZAR - VERIFICAR ESTADO
-------------------------------------
  sudo batctl if              --> debe mostrar "wlo1: active"
  lsmod | grep batman         --> debe mostrar "batman_adv"
  ip addr show bat0           --> ver IP actual de bat0


================================================================
PASO 1: LEVANTAR LA RED MESH (Nodo 1 - este PC)
================================================================

  sudo bash /home/julian/ProyectoSC/scripts/setup_adhoc_now.sh

  Si falla algún paso, hazlo manualmente:

  a) Desconectar WiFi normal:
       sudo nmcli device disconnect wlo1

  b) Bajar interfaz:
       sudo ip link set wlo1 down

  c) Poner en modo Ad-Hoc:
       sudo iw wlo1 set type ibss
       sudo ip link set wlo1 up

  d) Unirse a la red ProyectoSC en canal 6:
       sudo iw wlo1 ibss join ProyectoSC 6

  e) Asignar IP a BATMAN:
       sudo ip addr flush dev bat0
       sudo ip addr add 192.168.200.101/24 dev bat0


================================================================
PASO 2: CONFIGURAR EL PC EXTRA (Nodo 2)
================================================================

  En el PC extra, hacer exactamente lo mismo pero con IP .102:

  sudo nmcli device disconnect wlo1
  sudo ip link set wlo1 down
  sudo iw wlo1 set type ibss
  sudo ip link set wlo1 up
  sudo iw wlo1 ibss join ProyectoSC 6
  sudo modprobe batman-adv
  sudo ip link add name bat0 type batadv
  sudo ip link set wlo1 master bat0
  sudo ip addr flush dev bat0
  sudo ip addr add 192.168.200.102/24 dev bat0
  sudo ip link set bat0 up

  NOTA: Si la interfaz WiFi no se llama wlo1, verificar con:
        ip link show

SOLUCIÓN DE PROBLEMAS COMUNES
================================================================

PROBLEMA: "iw: command not found"
  SOLUCIÓN: sudo apt install iw

PROBLEMA: "batman_adv module not found"
  SOLUCIÓN: sudo apt install batctl
            sudo modprobe batman-adv

PROBLEMA: "RTNETLINK answers: Operation not supported" al hacer ibss join
  POSIBLE CAUSA: El driver WiFi no soporta modo Ad-Hoc
  VERIFICAR: iw phy phy0 info | grep -A5 "Supported interface modes"
             debe mostrar "IBSS" en la lista

PROBLEMA: bat0 no aparece
  SOLUCIÓN: sudo modprobe batman-adv
            sudo ip link add name bat0 type batadv
            sudo ip link set wlo1 master bat0

PROBLEMA: ping entre nodos no funciona
  VERIFICAR:
    - Ambos en el mismo canal (6)
    - Mismo SSID (ProyectoSC)
    - IPs distintas (.101 y .102)
    - sudo batctl n  (ver si se ven como vecinos)

PROBLEMA: Flask no inicia / puerto ocupado
  SOLUCIÓN: DASHBOARD_PORT=5001 python3 web_dashboard/app.py

PROBLEMA: "ModuleNotFoundError: flask"
  SOLUCIÓN: pip3 install flask
